# 📊 초등학생을 위한 막대그래프 보고서 만들기

초등학교 4학년 3학기 수학 교과 과정에 적합하게 개발된 **"자료 조사 및 막대그래프 보고서 만들기"** 실시간 웹 애플리케이션입니다.
학생들이 웹 브라우저를 통해 직접 통계 자료를 표에 입력하고, 막대 방향 및 눈금 크기를 정밀 조정하고, 색상 및 두께를 꾸민 뒤, 인쇄 가능한 최종 수학 탐구 보고서 형태로 이미지 저장 및 패들릿 공유를 할 수 있도록 설계되었습니다.

---

## 🚀 GitHub 및 Vercel을 통한 실시간 웹 배포 가이드

본 프로젝트는 데이터베이스가 필요 없는 **순수 정적 웹사이트(HTML/CSS/JS)**이므로, GitHub와 Vercel을 연동하여 1분 안에 전 세계 누구나 접속할 수 있는 무료 도메인으로 배포할 수 있습니다.

### 1단계: GitHub에 코드 올리기

1. **GitHub 계정 로그인 및 저장소 생성**
   - [GitHub](https://github.com)에 로그인합니다.
   - 우측 상단의 **`+`** 버튼을 누르고 **`New repository`**를 선택합니다.
   - **Repository name**에 원하는 이름(예: `graph-report`)을 입력합니다.
   - 공개 여부(Public)를 선택한 후, 아래의 **`Create repository`** 버튼을 클릭합니다.

2. **파일 업로드 (가장 쉬운 웹 브라우저 방법)**
   - 저장소가 생성되면 나오는 첫 화면에서 **`uploading an existing file`**이라는 문구를 클릭합니다.
   - 내 컴퓨터의 `graph-report` 폴더에 있는 아래 4개 파일을 드래그 앤 드롭으로 웹 브라우저 창에 올려놓습니다:
     - `index.html`
     - `style.css`
     - `script.js`
     - `.gitignore`
   - 파일 업로드가 완료되면 하단의 **`Commit changes`** 초록색 버튼을 클릭합니다.
   - 이제 GitHub 저장소에 코드가 안전하게 업로드되었습니다!

---

### 2단계: Vercel을 통해 무료 웹 주소로 배포하기

[Vercel](https://vercel.com)은 GitHub에 저장된 정적 파일을 자동으로 감지하여 무료 고속 웹 서버로 올려주는 호스팅 서비스입니다.

1. **Vercel 가입 및 로그인**
   - [Vercel 공식 홈페이지](https://vercel.com)에 접속합니다.
   - **`Sign Up`**을 누르고, **`Continue with GitHub`**를 클릭하여 내 GitHub 계정으로 가입/로그인합니다.

2. **프로젝트 가져오기 (Import)**
   - Vercel 대시보드 화면에서 우측 상단의 **`Add New...`** -> **`Project`**를 클릭합니다.
   - 조금 전 생성한 GitHub 저장소 목록이 화면에 나타납니다. `graph-report` 저장소 옆의 **`Import`** 버튼을 클릭합니다.

3. **즉시 배포 완료 (Deploy)**
   - **Configure Project** 설정 화면이 나타납니다.
   - *이 프로젝트는 빌드 과정이 없는 완전 정적 파일 구조이므로 별도의 Build & Development Settings를 수정할 필요가 없습니다.*
   - 하단의 **`Deploy`** 버튼을 클릭합니다.
   - 10~20초 동안 애니메이션이 실행된 후 **"Congratulations!"** 폭죽 축하 화면이 나타나면 배포가 완전히 끝난 것입니다!

4. **웹사이트 도메인 확인**
   - 화면에 표시된 이미지(미리보기 화면)를 클릭하거나 **`Visit`** 버튼을 누르면 `https://your-project-name.vercel.app` 형태의 고유 주소로 서비스 중인 실시간 웹 앱에 바로 접속할 수 있습니다.
   - 이 주소를 학급 커뮤니티(클래스팅, e학습터 등) 또는 학생들의 태블릿 PC 브라우저 주소창에 공유해 주시면 수업 시간에 즉시 사용하실 수 있습니다!

## 📝 교사 커스텀 설정 (학급 패들릿 주소 바꾸기)

학생들이 작성한 보고서를 학급 패들릿 게시판에 바로 제출할 수 있게 링크가 구현되어 있습니다.

1. `script.js` 파일을 메모장이나 에디터로 엽니다.
2. 파일의 **10번째 라인** 부근에 있는 `PADLET_URL` 변수의 주소를 선생님 학급의 패들릿 주소로 교체합니다:
   ```javascript
   const PADLET_URL = "https://padlet.com/선생님계정/선생님학급패들릿";
   ```
3. 수정 후 저장하고 GitHub 저장소에 파일을 덮어쓰기하여 커밋하면, **Vercel이 자동으로 변경 내용을 감지하고 5초 안에 웹사이트 주소에 반영(CI/CD)**해 줍니다!
